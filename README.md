# Default+

A dark theme inspired by Xcode's default dark color scheme, available for multiple editors and terminals.

## Overview

Default+ is a carefully crafted dark theme that brings Xcode's familiar color palette to your favorite development tools. The theme features a dark background with vibrant syntax highlighting that reduces eye strain during long coding sessions.

## Supported Applications

- **Nova Editor** - Theme extension
- **iTerm** - Color scheme configuration
- **Apple Terminal** - Terminal profile
- **Xcode** - Color theme
